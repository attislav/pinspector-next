Pin Inspector Bookmarklet

Ziehe den folgenden Button in deine Lesezeichenleiste:

Pin Inspector

Anleitung:

  1. Zeige deine Lesezeichenleiste an (Strg+Shift+B)
  2. Ziehe den roten Button oben in die Lesezeichenleiste
  3. Gehe auf eine beliebige Pinterest Pin-Seite
  4. Klicke auf "Pin Inspector" in der Lesezeichenleiste
  5. Ein neuer Tab mit allen Pin-Daten und Annotations wird geoeffnet

Was du siehst:

  • Pin-Bild, Titel, Description
  • Annotations als klickbare Chips (verlinkt zur Ideas-Seite)
  • Board, Pinner, Domain, Erstelldatum
  • Saves, Repins, Comments
  • Link zum Originalartikel
  • Rich Pin Metadata
