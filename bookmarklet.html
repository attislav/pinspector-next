Pin Inspector Bookmarklets

Overlay Bookmarklet Empfohlen

Zeigt Pin-Daten direkt als Overlay auf der Pinterest-Seite an:

Pin Inspector Overlay

So funktioniert's:

  1. Ziehe den roten Button in deine Lesezeichenleiste (Strg+Shift+B)
  2. Gehe auf eine beliebige Pinterest Pin-Seite
  3. Klicke auf "Pin Inspector Overlay"
  4. Ein Panel erscheint rechts mit allen Pin-Daten
  5. Nochmal klicken zum Schliessen, oder X-Button im Panel

Neuer-Tab Bookmarklet

Oeffnet Pin-Daten in einem neuen Tab (falls Overlay nicht funktioniert):

Pin Inspector (neuer Tab)

Was du siehst:

  • Pin-Titel, Description, Stats (Saves, Repins, Comments)
  • Annotations als klickbare Chips
  • Board, Pinner, Domain, Erstelldatum
  • Link zum Originalartikel
  • Rich Pin Metadata
